Keyboard Shortcuts

There are keyboard shortcuts for a few specific Structure Window : View commands in Cn3D. These shortcuts are not available on all platforms, but the equivalent menu items are always present.

 

Keyboard shortcuts in the Structure Window
Zoom in Z
Zoom out X
Show next frame Right Arrow
Show previous frame Left Arrow
Show first frame Down Arrow
Show last frame Up Arrow
Show all frames A
Animate: play frames P
Animate: spin N
Stop animation S

 

Mouse Actions

The following tables tell how to interact with Cn3D using the mouse and associated key presses. There are two tables: one for the structure window, and one for sequence/import windows.

"Left-drag" here means hold down the left button (or the only button, on a Macintosh) and drag the mouse. "Shift + drag" means hold down the 'shift' key while dragging the mouse. "Ctrl/Meta" means that on some platforms, the 'control' key is used (PC and UNIX), and on others the 'meta' key is used (Macintosh).

 

Mouse actions in the Structure Window
Rotate Left-drag
Translate Shift + left-drag
Zoom Ctrl + left-drag
Select residue Double-click
Set rotation center Ctrl/Meta + double-click

 

Mouse actions in Sequence and Import Windows
Set selection Left-drag
Extend selection Shift + left-drag
Invert selection Ctrl/Meta + left-drag
Browse Entrez Left-click on title/identifier
