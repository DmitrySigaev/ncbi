Most of the executables listed below are built on a regular basis, at least once a day from the latest sources, and if you are in NCBI, then you can find the latest version in directory:
$NCBI/c++/Release/bin/ (or $NCBI/c++/Debug/bin/).


Tool Synopsis
datatool Using ASN.1 and XML data specifications, generate C++ classes to serialize (read/write) the data
lbsmd New load-balancing service mapping daemon and related utilities
dispd.cgi Network service mapper and dispatcher
fwdaemon NCBI firewall daemon
