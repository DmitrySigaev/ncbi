Object Manager

Object Manager is a library of C++ classes, which facilitate access to biological sequence data. It makes it possible to transparently download data from the GenBank database, investigate bio sequence data structure, retrieve sequence data, descriptions and annotations.


This project is currently under development. The document is published here for reference only. You are welcome to post any comments to authors. As for now, we do not encourage you to actually use this library for your own developement.

Mailing list: cpp-objmgr@ncbi.nlm.nih.gov
Authors:
