Deprecated MSVC++ IDE configuration

In order to adjust the MSVC++ project files for your local C:\cxx you must do exactly the following after launching MSVC++ Developer Studio:
  • go to "Tools"/"Options..." form
  • goto folder "Directories"
  • choose option "Include files" in the "Show directories for:" option menu
  • add "C:\cxx\include" to the shown include directory list
  • if you want to use "internal" parts of the NCBI C++ toolkit (such as xhc library) then add "C:\cxx\include\internal" to the shown include directory list
