Using CVS on Mac OS in NCBI

Setting up your account on the server

Before you can successfully use a CVS client, you must set up your account on ray, the CVS server. The statements beginning with '$' are shell commands that you should enter. Occurances of parameter need to be replaced. The character '#' begins a comment, which should be omitted.

  1. Log into ray.
  2. $ echo hostname username >> .rhosts
  3. $ chmod 600 .rhosts
  4. Edit the file .ncbi_hints. On the line starting with "facilities", append "ncbi_tools".
  5. Log into ray again.

Installing and Configuring MacCVS Pro

  1. Download MacCVS Pro 2.5b2 (the latest release version as of 2001-03-14) from http://www.maccvs.org/. Although it's labeled as a beta, it's been used for years and the maintainers have not found an update necessary.
  2. Put the expanded folder somewhere appropriate, e.g. with other development tools. Launch the MacCVS Pro application. (It's not necessary to use the debug version.)
  3. Create a new session document by choosing [File->New] and saving the document.
  4. Choose [Edit->Session Settings...]:
    • Checkout and Update Options
      • Local Tree Directory: Click Set and choose a folder in which to check out a CVS working directory.
      • Merge Policy: Select Auto Merge Text Files and Update Binary Files.
    • Remote Host Information
      • CVS Server Settings
        • Server Hostname: ray
        • CVS Root: /src/NCBI/vault.ncbi
      • Authentication Method: RSH
        • CVS User Name: username
    • Messages and Misc.
      • Message Output
        • Checking Automatically clear message window is recommended, though optional.
    • Click OK.
  5. Choose [Action->Check Out Module...]. Enter the module name and click Check Out.
