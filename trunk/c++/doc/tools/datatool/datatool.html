DATATOOL -- the data serialization utility (NCBI C++)

Location c++/src/serial/tool
Synopsis Using ASN.1 and XML data specifications, generate C++ classes to serialize (read/write) the data

This program is intended for data conversion between ASN.1 and XML and for generation of C++ class hierarchy based on ASN.1 or XML data definition.
