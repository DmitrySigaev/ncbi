Locating the Class Definitions

In order to traverse an ASN data object, you must become familiar with the class structures and access functions which implement the object. The example provided here involves an ASN.1 Biostruc type which is implemented by class CBiostruc.

The first question then, is: how do I locate the class definitions implementing the object to be traversed? As mentioned previously, the source browser is of little help in this case, as the compiled object classes are not registered in the C++ tree. The include and src directories are available for browsing however, in $NCBI/c++/include/objects and $NCBI/c++/src/objects.

Because we are using a Biostruc, the first place to search is in the mmdb directories. In the mmdb1 directory, we find two classes: Biostruc.hpp and Biostruc_Base.hpp. As implied by the class names, CBiostruc inherits from CBiostruc_Base.
