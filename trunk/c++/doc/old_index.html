C++/STL Developer Guide for NCBI SWG

C++ is feature rich language, unlike C. It's sort of like a swiss army knife to C's buck knife. Misuse can cause a lot of trouble for both the programmer and whoever subsequently uses their code.

This guide is to help one to program on C++/STL in NCBI SWG and to provide a uniform appearance and functionality of the in-house code. Please read, think upon and follow these rules.

Note that there is still a lot of yet unresolved questions on the use of C++/STL in NCBI. Should you have any ideas, thoughts or proposals, please send them to the NCBI C++ group. Send there your own questions as well. Also, feel free to add your questions/answers to this page via CVS (checkout, edit and commit "internal/cpp/doc/todo.html").
