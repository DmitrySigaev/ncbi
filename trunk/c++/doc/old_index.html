C++/STL Developer Guide for NCBI SWG

  • CVS location: internal/c++/doc
  • Public WWW docs (the latest version): http://www.ncbi.nlm.nih.gov/IEB/CPP_DOC/index.html

This guide is intended to support C++/STL projects developed in the NCBI SWG. Guidelines for generating uniform appearance and functionality of in-house code are given. Please read, think upon and follow these rules.
