Building the whole NCBI C++ tree locally

We have built the NCBI C++ toolkit with Metrowerks CodeWarrior Pro 6. Other compilers or versions of CodeWarrior have not been tested.

The build script is currently using a hard-coded search path for the source tree.

  • The original location requires renaming c++ to cxx and moving it to the folder Metrowerks CodeWarrior 6.0 at the root level of the startup disk. (Whether this is actually an installation of CodeWarrior or an otherwise empty folder doesn't matter, as long as you have CodeWarrior installed somewhere.)
  • An alternate location is the following: <startup-disk>:Development:Trees:NCBI:ncbi.vault:internal:c++. In this case, any of these folders can may be replaced with an alias to a folder. This arrangement is intended to complement the use of a Mac CVS client.

To build the toolkit libraries and test programs:

  1. Open the folder mac_prj, which is in the folder compilers (inside c++).
  2. Open the file makelibs.met with Script Editor (e.g. by dragging the file's icon onto an icon for Script Editor). Click Run. There may be some compiler warnings in CodeWarrior, but this is not a problem. A beep signals completion of the script.
  3. Then do the same thing with maketests.met.
