How to run preinstalled Genome Workbench in NCBI

NCBI keeps Genome Workbench preinstalled on Unix servers in $NCBI/bin directory.
Please use $NCBI/bin/gbench to run it.
For Windows users Genome Workbench is available on DIZZY. The network path is: \\DIZZY\public\cxx\bin\gbench\bin\gbench.exe
