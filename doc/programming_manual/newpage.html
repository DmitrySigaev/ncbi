For example, we could implement the Visit function on a CBiostruc::TDescr as follows: 

void Visit (const CBiostruc::TDescr& descList) { 

    for (CBiostruc::TDescr::const_iterator i1 = descList.begin();
         i1 != descList.end(); ++i1) {

        for (CStdTypeConstIterator<string> i = ConstBegin(**i1); i; ++i) {
            cout << *i << endl;
        }
    }
}
The type iterators come in two flavors, CStdTypeIterator and CTypeIterator, with corresponding const implementations. This example uses a const standard type iterator to visit all string data members in a CBiostruc_descr object. Alternatively, if we had wanted to visit only the CBiostruc_history descriptors, the inner loop could be written as: 

   for (CTypeConstIterator<CBiostruc_history> i = ConstBegin(**i1); i; ++i) {
       visit(*i);
   }

In both cases, the type iterators use type information about the object's class to walk through its list of data members, stopping at only those of the specified type. previous     up
