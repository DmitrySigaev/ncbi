NCBI C++ Library Dependency Graph

This graph was obtained by building the full toolkit and then running the following commands: 
.../bin/depgraph .../lib/lib* > depgraph.out
.../bin/depgraph2dot -simplify -in depgraph.out -out depgraph.dot
dot -Tgif -o depgraph.gif.tmp depgraph.dot
giftopnm depgraph.gif.tmp | pnmtoppm '#FFFFFF' | ppmtogif -transparent '#FFFFFF' > depgraph.gif
where dot comes from AT&T's Graphviz package. If you are at NCBI, you can find the source to depgraph and depgraph2dot here.
