Object Manager.

Object Manager is a library of C++ classes, which facilitate access to biological sequence data. It makes it possible to transparently download data from the GenBank database, investigate bio sequence data structure, retrieve sequence data, descriptions and annotations.

Authors:
Andrei Gourianov, Aleksey Grichenko, Michael Kimelman, Denis Vakatov.
