Object Manager

Internal classes - definition

Here we define Object Manager's internal classes and their behavior. Internal classes should never be used by a client program directly.

  • Seq_id handle
  • Seq_id mapper
  • Bioseq info
  • Top-level seq entry info
  • Annotation object
  • Annotation iterator (TSE)
  • Handle range
  • Handle range map

Seq_id handle

Seq_id handle is an internal object, which is used as an alias (or substitute) to Seq_id. Due to complex nature of Seq_id, it is replaced by this lightweight object.

Seq_id mapper

Seq_id mapper maps Seq_id to Seq_id handle, back and forth. Such map is unique for each Object Manager - hence it is created and maintained by it. Different Object Managers have different maps. The map is populated when the data is loaded from various data sources. For each Seq_id the mapper creates the handle if necessary.

Bioseq info

Bioseq info object stores the set of  Seq_id handles which identify the biological sequence, as well as a reference to the Seq_entry, which contains the sequence. Since all such Seq_id handles refer to the same bio sequence, they are called synonyms here.

Top-level seq entry info

Top-level seq entry (TSE) info object stores a map of Seq_id handles to corresponding Bioseq info objects as well as a reference to the TSE which it describes. Another information stored here is a map of  Seq_id handles to annotation maps which list all annotation objects for the sequence.

Annotation object

Annotation object is a substitute to Seq_annot. It is used to facilitate the search for annotations which refer to a proper location on the sequence.

Annotation iterator (TSE)

While "regular" annotation iterator enumerates all annotations for a given sequence in the Scope, this iterator traverses only annotations within a given top-level Seq_entry. In fact "regular" iterator uses this one in its work.

Handle range

Handle range contains list of ranges (CRange) for a particular Bioseq handle. It facilitates the search for annotations on the bio sequence.

Handle range map

Handle range map maps Bioseq handle to Handle range. It serves as a substitution for Seq_loc and is used by annotation iterators
