NCBI C++ libraries

 
Library Project Synopsis & Hints
xncbi corelib Provide a portable low-level API and a lot of useful classes, functions and macro. 
HINT: It must be linked to every executable which uses the NCBI C++ toolkit! 
xconnect
connect
 Connection library. Includes platform-independent socket interface, SOCK, connection object, CONN, and specialized connector constructors, socket, file, HTTP and service, to be used as engines for connections. Also, provides access to load-balancing daemon and NCBI named service dispatching facilities.
xutil
util
 Collection of useful classes which can be used in more then one application.
xcgi cgi An integrated framework to write CGI applications: helps to retrieve and parse HTTP request, and then to compose and send out HTTP response. 
xfcgi cgi A Fast-CGI version of xcgi. 
HINT: Requires the target executable to be linked with a 3rd-party FastCGI library, as in: LIBS= $(FASTCGI_LIBS). 
HINT: On non-FastCGI capable platforms (or if run as a plain CGI on a a FastCGI capable platform), it works just as a plain CGI. 
xhtml html Compose and printout a HTML page -- using a static HTML template with embedded dynamic fragments. Provide a rich set of classes to help build the dynamic fragments of regular HTML tags (and text) arranged into a tree-like structure. 
xserial serial Generic library to provide data serialization in different formats. See also datatool. 
dbapi_driver dbapi Driver API for the NCBI database connectivity project.
